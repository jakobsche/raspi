TGPIOPort.ActiveLow

relationship between the boolean value of the property Value and the hardware port state

published property TGPIOPort.ActiveLow : Boolean
  read GetActiveLow
  write SetActiveLow;

Description

if ActiveLow = False then Value = False means high voltage and Value = True means low voltage at the hardware port. ActiveLow = True exchanges the voltages at the ports related to the property Value.
